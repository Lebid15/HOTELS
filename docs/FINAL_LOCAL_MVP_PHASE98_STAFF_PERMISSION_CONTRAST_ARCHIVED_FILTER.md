# Phase 98 — Staff Permission Contrast + Archived Filter

## الهدف
إغلاق ملاحظتين بصريتين/تشغيليتين داخل صفحة الموظفين بعد Phase 97:

1. شرائح الصلاحيات داخل مودالات الموظفين كان نصها أبيض فوق خلفية فاتحة، لذلك أصبحت غير واضحة.
2. فلتر الحالة لم يكن يعرض خيار الموظفين المؤرشفين، رغم وجود حالة `archived` داخل بيانات الموظفين.

## ما تم تنفيذه

- تقوية خلفية شرائح الصلاحيات داخل مودالات الموظفين باستخدام تدرج مركزي واضح.
- الحفاظ على النص الأبيض داخل شرائح الصلاحيات ولكن فوق خلفية مناسبة.
- تمييز الصلاحيات المحددة بتدرج مختلف أوضح.
- إضافة حالة `archived` إلى قائمة حالات الموظفين في صفحة الموظفين.
- تعديل جلب بيانات الموظفين بحيث يتم تضمين المؤرشفين فقط عند اختيار فلتر `مؤرشف`، بينما يبقى عرض `الكل` بدون مؤرشفين حتى لا تختلط السجلات المؤرشفة مع التشغيل اليومي.
- إضافة تمييز بصري لكرت الموظف المؤرشف عند عرضه.

## ملاحظات الأمان

- لم يتم تغيير منطق إنشاء الموظف أو تعديله.
- لم يتم تغيير صلاحيات الموظفين نفسها.
- لم يتم تعديل أي صفحة خارج صفحة الموظفين.
- تمت المحافظة على نظام المركزية الخاص بصفحة الموظفين من Phase 97.
