# Final Local MVP Closure — Phase 43

## Current Subscription Compact Panel

تم تصغير قسم الباقة الحالية حتى لا يزاحم كروت الباقات.

## المطلوب

كروت الباقات يجب أن تكون واضحة وأساسية في صفحة الاشتراك والباقات، بينما تفاصيل الباقة الحالية تظهر بشكل مختصر أسفلها.

## ما تم

- تصغير كرت الباقة الحالية.
- تصغير أيقونة الباقة الحالية.
- تقليل ارتفاع حقول البداية والنهاية والسعر وحالة الدفع.
- تصغير رسالة سياسة التجديد والتمديد.
- إخفاء مزايا الباقة داخل كرت الباقة الحالية لأنها تظهر داخل كروت الباقات.
