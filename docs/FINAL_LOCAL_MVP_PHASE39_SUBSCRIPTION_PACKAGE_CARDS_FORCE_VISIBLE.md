# Final Local MVP Closure — Phase 39

## Subscription Package Cards Force Visible

تم تصحيح صفحة الاشتراك والباقات بحيث تظهر كروت الباقات مباشرة في أعلى الصفحة.

## المشكلة

رغم نقل قسم الباقات، بقيت كروت الباقات لا تظهر للمستخدم بالشكل المطلوب بسبب:
- اعتماد العرض على الباقات الفعالة فقط.
- ترتيب الباقات بعد كروت الملخص.
- عدم ضمان ظهور الباقة الحالية إن لم تكن ضمن قائمة الباقات الفعالة.

## الحل

تم تعديل:

```text
apps/web/public/assets/js/modules/11a-subscription-plan.js
```

بحيث أصبح ترتيب الصفحة:

```text
عنوان الصفحة
كروت الباقات المتاحة
كروت ملخص الاشتراك
تفاصيل الباقة الحالية + طلبات الاشتراك
```

كما تمت إضافة:

```text
getVisibleSubscriptionPackages()
```

لتعرض:
- الباقات الفعالة.
- أو الباقات غير المؤرشفة إذا لم توجد باقات فعالة.
- والباقة الحالية دائمًا إذا لم تكن موجودة ضمن القائمة.

npm run workspace-subscription-packages:audit
