# Phase 96 — Rooms Floor Badge Contrast Fix

## الهدف
إصلاح ملاحظة أن دائرة رقم الطابق داخل صفحة الغرف والطوابق باهتة وغير واضحة.

## التعديل
- تقوية خلفية شارة رقم الطابق باستخدام تدرج النظام المركزي.
- جعل الرقم أبيض وواضحًا مع وزن خط قوي.
- تحسين الحد والظل بدون استخدام قيم عشوائية خارج Design Tokens.
- تطبيق الإصلاح على:
  - `room-floor-count`
  - `floor-overview-badge`

## النطاق
التعديل محصور في صفحة الغرف والطوابق فقط ولا يغير منطق البيانات أو المودالات أو الفلاتر.

## الفحص المطلوب
- `npm run quality:full`
