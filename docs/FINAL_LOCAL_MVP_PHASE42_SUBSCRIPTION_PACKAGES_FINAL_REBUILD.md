# Final Local MVP Closure — Phase 42

## Subscription Packages Final Rebuild

تم تثبيت صفحة الاشتراك والباقات حسب المطلوب النهائي: الصفحة يجب أن تكون صفحة باقات أولًا.

## الترتيب النهائي

```text
عنوان الصفحة
كروت الباقات المتاحة مباشرة
تفاصيل الباقة الحالية + طلبات الاشتراك
```

## ما تم تأكيده

- لا توجد كروت ملخص مكررة داخل الصفحة.
- كروت الباقات تستخدم `subscription-offer-card` و `subscription-offers-grid`.
- الباقة الحالية تظهر ضمن الكروت حتى لو لم تكن ضمن قائمة الباقات الفعالة.
- عند عدم وجود باقات فعالة، يتم الرجوع للباقات المسجلة غير المؤرشفة.
- التجديد والتمديد ممنوعان إذا كان الاشتراك فعالًا خارج آخر 3 أيام.
- تغيير الباقة متاح دائمًا.
