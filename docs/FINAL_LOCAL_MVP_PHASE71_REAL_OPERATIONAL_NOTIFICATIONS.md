# Final Local MVP Phase 71 — Real Operational Notifications

## الهدف
إعادة بناء صفحة الإشعارات لتصبح صفحة تنبيهات تشغيلية حقيقية للفندق، بحيث تعرض ما يحدث داخل الفندق وتتحكم بحالة القراءة وعدّاد الجرس بشكل منطقي.

## ما تم تنفيذه
- اعتماد إشعارات مولدة من بيانات التشغيل الحالية مثل:
  - وصول اليوم.
  - مغادرة اليوم.
  - مبالغ مستحقة.
  - غرف تحت التنظيف.
  - بلاغات صيانة مفتوحة.
  - طلبات على حساب الغرف.
  - إشعارات المنصة لصاحب المنصة مثل الاشتراكات والفنادق الموقوفة.
- إضافة حالة قراءة لكل إشعار عبر localStorage.
- عداد الجرس صار يحسب فقط الإشعارات غير المقروءة.
- فتح إشعار من الجرس أو من الصفحة يعلّمه كمقروء ويزيله من عداد الجرس.
- إعادة بناء صفحة الإشعارات كصفحة قراءة وتنبيه فقط، بدون نماذج أو CRUD معقد.
- إضافة فلاتر: الكل / غير مقروء / مقروء / عاجل غير مقروء.
- إضافة زر تعليم الكل كمقروء.
- تحسين الشكل البصري لكروت الإشعارات وتمييز المقروء وغير المقروء.
