# Phase 95 — Rooms Visual Contrast & Action Row Fix

## Scope
Targeted visual stabilization for the already-centralized **Rooms & Floors** page after Phase 94.

## Fixes
- Improved contrast of rooms summary metric cards.
- Forced summary card icons, values, labels, and notes to render with high contrast on gradient backgrounds.
- Kept metric card tones diverse and semantic: accent, luxury, success, warning, danger.
- Changed room card actions to a three-column one-row action layout for `View / Edit / Archive`.
- Avoided broad global selectors to protect previous stable pages.

## Safety
- No business logic changes.
- No room/floor data changes.
- No modal behavior changes.
- No sidebar/topbar changes.
