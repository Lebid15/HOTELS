# Final Local MVP Closure — Phase 46

## Platform Package Offer Cards

تم إضافة كروت عروض الباقات القادمة من لوحة صاحب المنصة داخل صفحة الاشتراك والباقات.

## المطلوب

بعد تفريغ الصفحة، يجب أن تحتوي فقط على كروت الباقات التي أضافها صاحب المنصة.

## مصدر البيانات

```text
fandqi.subscriptionPackages
```

## ما يظهر في الصفحة

- كروت الباقات غير المؤرشفة فقط.
- اسم الباقة.
- وصف الباقة.
- السعر والعملة.
- مدة الاشتراك.
- عدد المستخدمين.
- عدد الغرف.
- دعم المطعم.
- دعم التقارير.
- دعم التجربة.
- زر طلب هذه الباقة عند كونها نشطة.
- تمييز الباقة الحالية إن وجدت.

## ما لا يظهر

- كروت الملخص.
- تفاصيل الباقة الحالية.
- طلبات الاشتراك.
- سياسة التجديد والتمديد.

## الفحص

```powershell
npm run workspace-subscription-packages:audit
npm run quality:full
```
